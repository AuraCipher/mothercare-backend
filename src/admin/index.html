Getting Started

Use an API client like Postman or cURL to log in and get your JWT token:

POST /auth/login
Content-Type: application/json

{
  "identifier": "admin",
  "password": "Admin@1234",
  "rememberMe": true
}

Copy the token value from the response, paste it above, and click Connect.
